A Simple, Capable, and Unified Interface for Process Spawning in Ruby

Tired of juggling between system, backticks, and Open3? Exeggutor provides one simple method that does it all - safely spawn processes with real-time output, captured results, and sane error handling.

files = run!(%W[ls -l]).stdout.split("\n")
puts "Found files: #{files}"

result = run!(%W[cp #{old_file} #{new_file}], can_fail: true) # Works even if old_file or new_file have spaces
raise "Failed to copy - is there enough disk space?" if !result.success?

result = run!(%W[run_build.sh], show_stderr: true) # Shows long build's output while also capturing it
output_path = result.stdout

Capturing Output
ruby
Copy
# Capture stdout only
result = run!(%W[ls -l], capture_stdout: true)

# Capture both streams
result = run!(%W[ls /nonexistent], capture_stdout: true, capture_stderr: true)
puts result.stderr # "ls: /nonexistent: No such file..."

# Get execution metadata
puts "Took #{result.duration.round(2)}s, exited #{result.exitcode}"
